# Assignment 2: Personal Portfolio Page

For the second assignment, I built a personal portfolio page using HTML, CSS, and Bootstrap 5 as requested in the syllabus.

## What I Did
- **Bootstrap 5 Setup:** Instead of writing every layout rule from scratch, I linked the Bootstrap 5 CDN to speed up development and ensure the site is responsive out of the box.
- **Sections:** I included a Navigation bar, a Hero section, About Me, Tech Stack/Skills, a Projects grid, and a Contact area.
- **Custom Styling:** Wrote a `style.css` file to override some default Bootstrap looks. I added smooth scrolling, card hover transitions, and a custom background image with an overlay for the header to make it look a bit more polished.
- **Icons:** Included FontAwesome via CDN so I could use nice standard icons for the social links and the tech stack.

## How to Check it out
Just open `index.html` in your web browser. Everything is loaded via public CDNs (Bootstrap, FontAwesome, and Google Fonts), so you just need an internet connection to see the full styles.

I hope the layout looks clean and checks off all the requirements!
