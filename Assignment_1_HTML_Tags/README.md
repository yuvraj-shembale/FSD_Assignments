# Assignment 1: HTML Text Formatting Tags

Hey there! This is my submission for the first FSD assignment. The goal here was pretty straightforward: create a static web page that displays all the basic HTML text formatting tags organized in a table.

I kept it simple by using a single `index.html` file with internal CSS for styling so we don't have to jump between files just to see the table styles. 

## What's included?
- A neat HTML table structure.
- Some basic CSS to make the table look clean (alternating row colors, a green header, hover effects).
- Examples of tags like `<b>`, `<strong>`, `<i>`, `<em>`, `<mark>`, `<small>`, `<del>`, `<ins>`, `<sub>`, and `<sup>`.

## How to run it
You don't need any server or complex setup for this one. Just double-click on `index.html` and it will open in your default browser.
