HTML Text Formatting Tags

Sr. No. Tag Name Description Output / Example
1 <b> Defines bold text without any extra importance. This text is bold
2 <strong> Defines text with strong importance (semantic bold). This text is strong
3 <i> Defines italic text without any extra importance. This text is italic
4 <em> Defines emphasized text, semantically important. This text is emphasized
5 <mark> Defines marked or highlighted text. This text is marked/highlighted
6 <small> Defines smaller text. This text is smaller
7 <del> Defines deleted (strike-through) text. This text is deleted
8 <ins> Defines inserted (underlined) text. This text is inserted
9 <sub> Defines subscript text (e.g., H₂O). H2O
10 <sup> Defines superscript text (e.g., x²). x2
11 <u> Defines text that should be stylistically different from normal text, like misspelled words. This text is underlined
